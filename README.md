## 简体中文

- 官网：<a href="https://bighelper.top" target="_blank">bighelper.top</a>
- 现已同时支持 PC 端与 Android 端，提供系统课程、AI 复盘、每日训练与对局辅助。
- 支持模式：四麻、三麻、修罗对战。
- 支持平台：雀魂（国服/日服/国际服）、麻雀一番街、天凤、掌心麻将、雀姬、天月麻雀。

### 功能特点

- 系统课程与每日训练，帮助建立稳定的牌理基础。
- AI 复盘即时指出错牌与更优线路，方便赛后总结。
- PC 与 Android 双端可用，最新安装说明与展示截图以官网为准。
- 成长档案、社区答疑与远程协助，帮助持续提升雀力。
