## 简体中文

- 官网：<a href="https://bighelper.top" target="_blank">bighelper.top</a>
- 支持网页与客户端运行：自动出牌、自动表情、对局托管；内置安全防封策略。
- 支持模式：四麻、三麻、修罗对战；现已支持麻雀一番街。
- 支持平台：雀魂（国服/日服/国际服）、麻雀一番街、掌心麻将、天凤。

### 功能特点

- 系统课程与每日训练，AI 复盘即时提示错牌与更优选择。
- 成长档案与数据追踪，稳步提升雀力。
- 跨平台支持与远程协助。
- 安全防封：多重风控策略与细粒度行为控制，尽量降低封禁风险。
- 自动换肤：根据环境自动切换皮肤/主题，减少手动设置成本。
