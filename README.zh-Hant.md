## 繁體中文

- 官網：<a href="https://bighelper.top" target="_blank">bighelper.top</a>
- 支援網頁與客戶端運行：自動出牌、自動表情、對局託管；內建安全防封策略。
- 支援模式：四麻、三麻、修羅對戰；現已支援麻雀一番街。
- 支援平台：雀魂（國服/日服/國際服）、麻雀一番街、掌心麻將、天鳳。

### 功能特色

- 系統課程與每日訓練，AI 複盤即時指出錯牌與更優選擇。
- 成長檔案與數據追蹤，穩步提升雀力。
- 跨平台支援與遠端協助。
