## 日本語

- 公式サイト：<a href="https://bighelper.top" target="_blank">bighelper.top</a>
- Web とクライアント両対応：自動打牌、スタンプ、自動対局管理。安全なアンチ BAN 対策を実装。
- 対応モード：四人麻雀、三人麻雀、修羅対戦。麻雀一番街も対応。
- 対応プラットフォーム：雀魂（国服/日服/国際版）、麻雀一番街、掌心麻雀、天鳳。

### 機能

- システム講座とデイリートレーニング、AI 解析でミス指摘と最善提案。
- 成長記録とデータ追跡で実力向上を支援。
- クロスプラットフォームとリモートサポート。
